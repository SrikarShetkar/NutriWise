# NutriWise – Full-Stack Nutrition & Budget Planning Platform

## Overview
NutriWise is a full-stack web application that generates personalized meal plans based on a user's BMI, calorie requirements, and budget. It helps users make healthier and cost-effective food choices while simplifying grocery planning.

## Features
- Personalized meal recommendations
- BMI and daily calorie analysis
- Budget-based meal planning
- Responsive user interface
- RESTful backend APIs
- Secure data storage

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Project Structure

```
frontend/

```

## Installation

```bash
git clone https://github.com/yourusername/NutriWise.git
cd NutriWise

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Start the application
npm start
```
